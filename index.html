📌 Come collegare il Google Sheet
1. Apri il Google Sheet → Estensioni → Apps Script
2. Incolla il codice e salva:
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet();
  var s = sheet.getSheetByName("Foglio1") || sheet.getSheets()[0];
  var data = s.getDataRange().getValues();
  var csv = data.map(function(r){
    return r.map(function(c){
      var v = String(c).replace(/"/g,'""');
      return v.indexOf(",")>=0||v.indexOf('"')>=0?'"'+v+'"':v;
    }).join(",");
  }).join("\n");
  return ContentService.createTextOutput(csv)
    .setMimeType(ContentService.MimeType.TEXT);
}
3. Distribuisci → Nuova distribuzione → Tipo: App web → Accesso: Chiunque
4. Copia l'URL e incollalo sopra.
